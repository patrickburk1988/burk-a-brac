A small but growing collection of miscellaneous tweaks for Foundry VTT.

## Settings

### 📌 Map Notes
| Setting:                       | Explanation:                                                               |
| ------------------------------ | -------------------------------------------------------------------------- |
| **Always Display Text Labels** | Force map notes to always display text labels.                             |
| **Use Image Aspect Ratio**     | Set map notes' dimensions to the same aspect ratio as their source images. |
